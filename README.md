# Building
1. clone this directory into your system via ``` git clone  https://github.com/Rank-N-Tensor/System_Monitor.git ```

2. move into directory 

3. ```mkdir build && cd build && cmake .. && make```

4.to run the application
```cd .. && ./build/monitor ```
